## kakao Exported chat analyzer

### 카카오톡 대화내용 분석기

### 기능
- 아이폰 내보내진 파일 자동 합치기
- 카카오톡 대화유저, 채팅 점유율, 입 퇴장, 선물 하거나 받은 횟수, 최근 채팅 일자등 csv파일 반환

### Support Tested

- Window Korean
- Iphone Korean
- Android English

### Unknown

- Window English
- Iphone English
- Android Korean
